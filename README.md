# Batas — RAG Q&A over the Philippine Labor Code

> Work in progress. Full README lands in Phase 6.

Ask questions about the Philippine Labor Code (PD 442, as amended & renumbered)
and get answers grounded in the exact articles, with verbatim citations and a
retrieval eval harness that proves accuracy with numbers.

Everything runs on free tiers: Supabase (pgvector + FTS), Gemini free tier
(embeddings + generation), Vercel Hobby, GitHub Actions.

**Educational demo — not legal advice.**
